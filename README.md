
# A/B-тестирование эффективности рекламной кампании

Учебный pet-проект, выполненный для отработки принципов **A/B-тестирования** и статистического анализа данных.
Цель — на примере случайных данных сравнить контрольную и тестовую группы пользователей и применить классические статистические методы для проверки гипотез.

## Основные шаги:
- Генерация искусственных данных для двух групп (Control и Variant)
- Проведение описательного анализа и визуализация распределений
- Проверка гипотез с помощью *t-теста*, *U-теста Манна–Уитни* и *z-теста для пропорций*
- Расчёт размера эффекта (Cohen’s d)

Данные в проекте **сгенерированы случайным образом**, поэтому результаты не имеют практической значимости — проект создан исключительно для демонстрации подходов к анализу данных и построению A/B-экспериментов.

**Технологии:** Python, pandas, NumPy, SciPy, Matplotlib, seaborn
